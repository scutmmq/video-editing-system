# 轻量级视频剪辑与多媒体处理系统

浏览器端视频处理工具，支持视频裁剪、GIF 转换、音频提取、文字水印、视频滤镜、封面截取。所有处理在本地浏览器中完成，无需上传文件到服务器。

## 环境要求

- **Node.js** 18 或更高版本（[下载地址](https://nodejs.org/zh-cn/)）

## 启动步骤

### 1. 解压文件

将压缩包解压到任意目录。

### 2. 安装依赖

在项目目录打开终端/命令提示符，运行：

```bash
npm install
```

> 如果下载速度慢，可以先用国内镜像：
> ```bash
> npm config set registry https://registry.npmmirror.com
> npm install
> ```

### 3. 启动服务器

```bash
node server.js
```

### 4. 打开浏览器

访问 **http://localhost:3000**

---

## Windows 快速启动

双击运行 `start.bat`，会自动安装依赖并启动服务器。

## 使用说明

1. 上传本地视频文件（支持 MP4、WebM、MOV、AVI、MKV）
2. 在视频预览区播放查看
3. 选择功能标签页，填写参数，点击处理
4. 处理完成后预览结果，点击下载保存到本地

## 功能列表

| 功能 | 说明 |
|------|------|
| 视频裁剪 | 截取指定时间段的视频片段 |
| GIF 转换 | 将视频片段转换为 GIF 动图 |
| 音频提取 | 从视频中提取 MP3/WAV 音频 |
| 文字水印 | 在视频上添加自定义文字 |
| 视频滤镜 | 黑白、模糊、亮度、对比度、镜像、复古、冷暖色调 |
| 封面截取 | 截取指定时间点的画面为 PNG 图片 |

## 注意事项

- 文件大小建议不超过 500MB，过大会导致处理时间较长
- 所有处理在浏览器本地完成，视频不会上传到任何服务器
- 需要现代浏览器（Chrome 96+、Edge 96+、Firefox 109+）
